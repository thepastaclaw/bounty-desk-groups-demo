Functional Testnet App

This section sends real Dash Platform testnet transitions from your browser. It creates a throwaway testnet wallet, asks you to fund it through Bridge, then registers identities, deploys the bounty contract, submits a token-paid claim, and runs 2-of-3 freeze/destroy actions.

live SDK
Testnet only. The generated mnemonic is stored in this browser's localStorage so the page can sign later steps. Do not use a real wallet or mainnet seed here. Reusing old identities may fail if previous runs froze or destroyed their token balance; generate a fresh wallet for a clean run.
